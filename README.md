## What can it do
- [x] Join online server
- [x] Join offline server
- [x] Parse chunk data
- [ ] View inventory
- [x] Use item
- [x] Listening sound
- [x] Send chat message
- [x] Recive and **parse** chat message
- [x] Move around and look around
- [x] Swing hands
- [x] Mine blocks
- [x] Place blocks
- [ ] Steer vehicle
- [ ] Steer boat
- [ ] Craft
- [ ] Pick item
- [x] Use entity
- [ ] Edit book
- [ ] Enchant item
- [ ] Record entity
- [ ] Unload chunks
- [ ] Path finding
